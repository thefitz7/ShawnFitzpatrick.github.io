    Honors and Awards

    Top Capstone Project


    Low Cost 3D Laser Scanner - COM DEV Sponsored

    Sept 2012 – Apr 2013

    Project was awarded the top project of the Electronics Engineering Technology - Telecommunications Systems program for 2013. The purpose of the low cost 3D scanner was to prove the idea that an object can be scanned into a system. Once it is scanned in, it can be measured and edited to form a fully rendered 3D representation. The object can then be compared to quality assurance regulations, to prove the quality of the said object. The system had an automated scanning process which moved the laser to scan the object, as well as rotated the object to required faces. The system was all ran through a user friendly interface which allowed users multiple options for scanning objects, such as selecting which faces to scan.
